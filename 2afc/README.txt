2afc_template
-------------------------------------------------------------------------------

Created by: Joseph V. Casillas
Last update: 8/31/2017

OVERVIEW

This is a PsychoPy2 implementation of two-alternative forced-choice task.

ABOUT

In this experiment the participant is presented stimuli randomly drawn from 
a /ba/-/da/ continuum. They are asked to decide if the stimuli sound more 
like "ba" or more like "da". The experiment is in English and provides feedback 
in real time. The script used to create the stimuli is provided in the 'stimuli' 
folder.

INSTRUCTIONS

Run the 2afc_template.psyexp file in PsychoPy2 or the 2afc_template.py file 
from coder.
