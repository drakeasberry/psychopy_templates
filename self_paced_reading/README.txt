Self-paced reading
-------------------------------------------------------------------------------

Created by: Joseph V. Casillas
Last update: 10/25/2017

OVERVIEW

A standard self-paced reading experiment implemented in PsychoPy2 builder mode. 

ABOUT

This is a generic self-paced reading task. It serves as a template that can 
easily be customized to fit the needs of a specific experiment.

INSTRUCTIONS

Run the self_paced_reading.psyexp file in PsychoPy2. To use longer sentences 
it is necessary to add a new routine for each additional word. See condition_file 
for details.
